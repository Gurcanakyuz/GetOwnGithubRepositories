# GetOwnGithubRepositories

You need a .env file in root. Using file cache system.

### Example .env file
```
# Github Username
USERNAME=ExampleUser

# Personal Access Token
PERSONAL_ACCESS_TOKEN=XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX

# Cache is Active/Deactive
CACHE=1
```

### Console Usage
> ./bin/list
